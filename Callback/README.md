# Callback in C

Esse projeto mostra o uso de callbacks em C de forma simples. Conceito simples mas poderoso para criar código flexível e reutilizável.

## Conceitos demonstrados:

| Conceito | Descrição |
|----------|-----------|
| Callback | Função passada como argumento para outra função |
| Ponteiros para funções | Variáveis que armazenam endereços de funções |
| Flexibilidade | Permite comportamento customizado sem modificar código existente |
| Reutilização | Mesma função de ordenação pode usar diferentes critérios de comparação |
